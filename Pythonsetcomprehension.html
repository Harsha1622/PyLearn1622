Set Comprehension in Python

What is Set Comprehension?

Set comprehension is a concise way to create sets using a single line of code. It is similar to list comprehension but uses curly braces { } to generate unique elements.

Syntax


{expression for item in iterable}

The expression is evaluated for each element in the iterable, and the results are stored as unique elements in a set.

Example: Creating a Set using Comprehension


numbers = {x for x in range(5)}

print(numbers)
{0, 1, 2, 3, 4}

Example with Condition


even_numbers = {x for x in range(10) if x % 2 == 0}

print(even_numbers)
{0, 2, 4, 6, 8}

Important Points

✔ Set comprehension creates sets in a compact form.
✔ Curly braces { } are used.
✔ Duplicate values are automatically removed.
✔ Conditions can be applied to filter elements.
