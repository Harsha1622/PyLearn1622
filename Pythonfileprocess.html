File Processing in Python

What is File Processing?

File processing refers to reading data from a file and performing operations such as counting, searching, or modifying the data. It is commonly used in applications that handle large amounts of stored information.

Reading File Line by Line

Instead of reading the entire file at once, Python allows reading a file line by line using a loop. 


file = open("data.txt","r")

for line in file:
    print(line)

file.close()

Example: Counting Number of Lines


file = open("data.txt","r")

count = 0
for line in file:
    count += 1

print("Total lines:", count)

file.close()
Total lines: 5

Example: Searching a Word in File


file = open("data.txt","r")

for line in file:
    if "Python" in line:
        print(line)

file.close()

Important Points

✔ File processing reads and analyzes file data.
✔ Loops are commonly used to process file content.
✔ Files can be searched, counted, or modified.
✔ Always close the file after processing.
