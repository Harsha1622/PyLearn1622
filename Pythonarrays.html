Arrays in Python

What is an Array?

An array is a data structure used to store multiple values of the same data type in a single variable. Arrays allow efficient storage and access to a collection of elements.

Array Module in Python

Python provides the array module to create arrays. Before using arrays, the module must be imported. 


import array

Creating an Array


import array

numbers = array.array('i', [10, 20, 30, 40])

print(numbers)
array('i', [10, 20, 30, 40])

Here 'i' represents integer type elements.

Common Array Type Codes

✔ 'i' → Integer
✔ 'f' → Floating point number
✔ 'd' → Double precision float
✔ 'b' → Signed integer

Advantages of Arrays

✔ Store multiple values in a single variable.
✔ Efficient memory usage.
✔ Easy access to elements using index.
✔ Useful for numerical data processing.

Example Program


import array

arr = array.array('i', [5, 10, 15, 20])

for i in arr:
    print(i)
