Tuple Slicing in Python

What is Tuple Slicing?

Tuple slicing is used to extract a portion of a tuple. It allows accessing multiple elements at once by specifying a range of indexes.

Syntax


tuple[start : end]

The start index is included, but the end index is excluded.

Example: Basic Tuple Slicing


numbers = (10, 20, 30, 40, 50)

print(numbers[1:4])
(20, 30, 40)

Slicing from Beginning


numbers = (10, 20, 30, 40, 50)

print(numbers[:3])
(10, 20, 30)

Slicing to the End


numbers = (10, 20, 30, 40, 50)

print(numbers[2:])
(30, 40, 50)

Important Points

✔ Tuple slicing extracts a range of elements.
✔ Start index is inclusive.
✔ End index is exclusive.
✔ Slicing does not modify the original tuple.
