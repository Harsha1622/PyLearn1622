Creation of Tuple in Python

Creating a Tuple

A tuple can be created by placing values inside parentheses () separated by commas. A tuple can store different types of data such as numbers, strings, and boolean values.

Syntax


tuple_name = (value1, value2, value3)

Example: Creating a Tuple


numbers = (10, 20, 30, 40)

print(numbers)
(10, 20, 30, 40)

Tuple with Different Data Types


data = ("Python", 3.11, True)

print(data)
('Python', 3.11, True)

Creating a Single Element Tuple

To create a tuple with only one element, a comma must be placed after the value. Without the comma, Python treats it as a normal variable. 


single = (10,)
print(single)
(10,)

Important Points

✔ Tuples are created using parentheses.
✔ Elements are separated by commas.
✔ Tuples can store multiple data types.
✔ A comma is required for single-element tuples.
