Tuples in Python

What is a Tuple?

A tuple is an ordered collection of elements in Python. Tuples are similar to lists but they are immutable, which means their values cannot be changed after creation.

Creating a Tuple


numbers = (10, 20, 30, 40)
print(numbers)
(10, 20, 30, 40)

Tuples are created using parentheses ().

Accessing Tuple Elements


numbers = (10, 20, 30, 40)

print(numbers[0])
print(numbers[2])
10
30

Tuple elements are accessed using indexing.

Tuple with Different Data Types


data = ("Python", 3.11, True)

print(data)
('Python', 3.11, True)

Tuples can store different types of values.

Important Points

✔ Tuples are ordered collections.
✔ Tuples are immutable (cannot be changed).
✔ Created using parentheses ().
✔ Support indexing and slicing.
