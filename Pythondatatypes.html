Data Types in Python

What is a Data Type?

A data type specifies the type of value that a variable can store. In Python, the data type is automatically determined based on the value assigned to the variable, so explicit declaration is not required.

1. Numeric Data Types

Numeric data types are used to store numerical values. Python supports integers, floating-point numbers, and complex numbers. 


a = 10      # integer
b = 3.5     # float
c = 2 + 3j  # complex number

2. Sequence Data Types

Sequence data types store collections of values in an ordered form. Examples include strings, lists, and tuples. 


lst = [1, 2, 3]
tup = (10, 20)
name = "Python"

3. Set Data Type

A set stores unordered and unique elements. Duplicate values are automatically removed. 


s = {1, 2, 2, 3}
print(s)

4. Mapping Data Type

Dictionaries store data as key–value pairs. Each key is unique and used to access its corresponding value. 


student = {"name": "Harsha", "language": "Python"}

5. Boolean Data Type

The Boolean data type represents logical values. It has only two possible values: True and False. 


x = True

6. None Data Type

The None data type represents the absence of a value. It is commonly used to initialize variables before assigning actual data. 


value = None
