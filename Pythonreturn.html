Return Statement in Python

What is Return Statement?

The return statement is used in a function to send a value back to the place where the function was called. It ends the execution of the function and returns the result.

Syntax of Return Statement


def function_name():
    statements
    return value

Example: Function with Return Value


def add(a, b):
    return a + b

result = add(10, 20)
print(result)
30

Example: Returning Multiple Values


def calculate(a, b):
    return a + b, a * b

sum_value, product = calculate(5, 3)

print(sum_value)
print(product)
8
15

Example: Return Without Value


def message():
    print("Hello Python")
    return

message()
Hello Python

Important Points

✔ The return statement sends a value back from a function.
✔ It stops the execution of the function.
✔ A function can return single or multiple values.
✔ If no return statement is used, the function returns None.
