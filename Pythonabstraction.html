Abstraction in Python

What is Abstraction?

Abstraction is the concept of hiding internal implementation details and showing only the necessary functionality to the user. It helps reduce program complexity and improves code readability.

Why Abstraction is Used?

✔ Hides complex implementation details
✔ Shows only essential features
✔ Improves program security
✔ Makes programs easier to maintain

Real-Life Example

When you drive a car, you use the steering wheel, brake, and accelerator. You do not need to understand the internal working of the engine. This is abstraction.

Abstraction using Abstract Classes

Python provides abstraction using the ABC (Abstract Base Class) module. 


from abc import ABC, abstractmethod

class Shape(ABC):

    @abstractmethod
    def area(self):
        pass

class Square(Shape):

    def area(self):
        print("Area of square")

s = Square()
s.area()
Area of square

Important Points

✔ Abstraction hides internal implementation details.
✔ It focuses on what an object does rather than how it does it.
✔ Python uses the abc module to create abstract classes.
✔ Abstract methods must be implemented in child classes.
