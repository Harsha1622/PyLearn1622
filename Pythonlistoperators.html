List Operators in Python

What are List Operators?

List operators are used to perform operations on lists such as combining lists, repeating elements, and checking membership.

1. Concatenation Operator (+)

The + operator joins two lists together. 


list1 = [1, 2, 3]
list2 = [4, 5]

result = list1 + list2
print(result)
[1, 2, 3, 4, 5]

2. Repetition Operator (*)

The * operator repeats list elements multiple times. 


numbers = [1, 2, 3]

print(numbers * 3)
[1, 2, 3, 1, 2, 3, 1, 2, 3]

3. Membership Operators

Membership operators check whether an element exists in a list. 


numbers = [10, 20, 30]

print(20 in numbers)
print(40 not in numbers)
True
True

4. Indexing Operator

Indexing is used to access elements of a list using position. 


numbers = [10, 20, 30, 40]

print(numbers[1])
20

Important Points

✔ + combines lists.
✔ * repeats list elements.
✔ in and not in check membership.
✔ Indexing accesses elements by position.
