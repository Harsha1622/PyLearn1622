Creation of Dictionary in Python

What is Dictionary Creation?

A dictionary can be created in Python by storing data in the form of key–value pairs using curly braces { }. Each key is associated with a value.

Creating Dictionary using Curly Braces


student = {
    "name": "Rahul",
    "age": 21,
    "course": "Python"
}

print(student)
{'name': 'Rahul', 'age': 21, 'course': 'Python'}

Creating an Empty Dictionary


student = {}

print(student)
{}

Creating Dictionary using dict() Function


student = dict(name="Rahul", age=21, course="Python")

print(student)
{'name': 'Rahul', 'age': 21, 'course': 'Python'}

Dictionary with Different Data Types


data = {
    "name": "Amit",
    "marks": 85,
    "passed": True
}

print(data)

Important Points

✔ Dictionaries are created using { }.
✔ Data is stored as key–value pairs.
✔ Keys must be unique.
✔ Values can be any data type.
✔ Dictionary can also be created using dict() function.
