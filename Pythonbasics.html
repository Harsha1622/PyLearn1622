Python Basics

Identifiers

Identifiers are names used to identify variables, functions, classes, modules, or other objects in a Python program. 


total_score = 85
user_name = "admin"

Rules for Identifiers:
✔ Letters, digits, and underscore (_) are allowed
✔ Must start with a letter or underscore
✔ Cannot start with a digit
✔ Special symbols are not allowed except underscore
✔ Python keywords cannot be used as identifiers 


2total      # ❌ Invalid
total_2     # ✅ Valid
total@value # ❌ Invalid

Keywords

Keywords are reserved words in Python that have predefined meanings. They cannot be used as variable names or identifiers. 


if user_logged_in:
    print("Access granted")

Variables

Variables are used to store data values. Python automatically determines the data type of a variable when a value is assigned. 


user_id = 1024
username = "developer"
account_balance = 2500.75


status = "active"
print(status)   # ✅ Correct
print(Status)   # ❌ Error

Indentation

Indentation is used to define code blocks in Python. Unlike many other programming languages, Python uses indentation instead of braces to structure code. The standard indentation level is four spaces. 


if account_balance > 0:
    print("Transaction allowed")
    print("Processing payment")

Comments

Comments are used to explain code and improve readability. They are ignored during program execution. 


# Display welcome message
print("Welcome to PyLearn")


'''
This program checks user access
and displays a message
'''
print("Access granted")
