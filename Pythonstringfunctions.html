String Functions in Python

What are String Functions?

String functions are built-in functions used to perform operations on strings such as finding length, checking data types, and converting values.

1. len() Function

The len() function returns the length of a string. 


text = "Python"

print(len(text))
6

2. str() Function

The str() function converts other data types into a string. 


number = 100

text = str(number)

print(text)
print(type(text))
100
<class 'str'>

3. int() Function

The int() function converts a string into an integer. 


text = "50"

num = int(text)

print(num)
50

4. float() Function

The float() function converts a string into a floating-point number. 


text = "10.5"

num = float(text)

print(num)
10.5

5. type() Function

The type() function is used to find the data type of a string. 


text = "Python"

print(type(text))
<class 'str'>

Important Points

✔ len() finds the length of a string.
✔ str() converts data to string.
✔ int() converts string to integer.
✔ float() converts string to float.
✔ type() identifies the data type.
