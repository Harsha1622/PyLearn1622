Accessing Array Elements in Python

Accessing Array Elements

Array elements can be accessed using an index value. The index starts from 0, which means the first element is stored at position 0.

Example: Accessing Elements


import array

arr = array.array('i', [10, 20, 30, 40])

print(arr[0])
print(arr[2])
10
30

Here arr[0] accesses the first element and arr[2] accesses the third element.

Accessing Elements using Loop


import array

arr = array.array('i', [5, 10, 15, 20])

for i in range(len(arr)):
    print(arr[i])
5
10
15
20

Accessing Elements using for-each Loop


import array

arr = array.array('i', [1, 2, 3, 4])

for element in arr:
    print(element)
1
2
3
4

Important Points

✔ Array indexing starts from 0.
✔ Elements can be accessed using index numbers.
✔ Loops can be used to access all elements.
✔ Index must be within array size.
