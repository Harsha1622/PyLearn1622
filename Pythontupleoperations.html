Tuple Operations in Python

Tuple Operations

Although tuples are immutable, Python allows several operations to be performed on tuples such as concatenation, repetition, membership testing, and length calculation.

Concatenation

Two tuples can be combined using the + operator. 


tuple1 = (1, 2, 3)
tuple2 = (4, 5)

result = tuple1 + tuple2
print(result)
(1, 2, 3, 4, 5)

Repetition

The * operator repeats tuple elements multiple times. 


numbers = (1, 2)

print(numbers * 3)
(1, 2, 1, 2, 1, 2)

Membership Operator

The in and not in operators check whether an element exists in a tuple. 


numbers = (10, 20, 30)

print(20 in numbers)
True

Length of Tuple

The len() function returns the number of elements present in a tuple. 


numbers = (10, 20, 30, 40)

print(len(numbers))
4

Important Points

✔ Tuples support concatenation using +.
✔ Tuples support repetition using *.
✔ Membership operators check element existence.
✔ len() returns the total number of elements.
