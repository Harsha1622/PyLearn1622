Strings in Python

What is a String?

A string is a sequence of characters used to represent text in Python. Strings are written inside single quotes (' ') or double quotes (" ").

Creating Strings


text1 = "Hello Python"
text2 = 'Welcome to PyLearn'

print(text1)
print(text2)
Hello Python
Welcome to PyLearn

Accessing Characters in a String

Characters in a string can be accessed using an index. The index starts from 0. 


text = "Python"

print(text[0])
print(text[3])
P
h

String Length

The len() function is used to find the length of a string. 


text = "Python"

print(len(text))
6

String Concatenation

Two or more strings can be combined using the + operator. 


first = "Hello"
second = "Python"

result = first + " " + second

print(result)
Hello Python

Important Points

✔ Strings store text data.
✔ Strings are immutable (cannot be changed).
✔ Indexing starts from 0.
✔ Strings support many built-in functions and methods.
