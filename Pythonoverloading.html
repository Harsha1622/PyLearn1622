Method Overloading and Method Overriding in Python

Method Overloading

Method overloading means defining multiple methods with the same name but different parameters. Python does not support traditional method overloading like some other languages, but it can be achieved using default arguments.

Example: Method Overloading


class Math:

    def add(self, a, b, c = 0):
        print(a + b + c)

m = Math()

m.add(5, 3)
m.add(5, 3, 2)
8
10

Here the same method add() works with different numbers of arguments.

Method Overriding

Method overriding occurs when a child class provides its own implementation of a method that already exists in the parent class.

Example: Method Overriding


class Animal:

    def sound(self):
        print("Animal makes sound")

class Dog(Animal):

    def sound(self):
        print("Dog barks")

d = Dog()
d.sound()
Dog barks

Important Points

✔ Method overloading uses the same method name with different parameters.
✔ Python handles overloading using default arguments.
✔ Method overriding occurs in inheritance.
✔ The child class replaces the parent class method implementation.
