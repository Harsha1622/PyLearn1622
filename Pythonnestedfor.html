Nested for Loop in Python

What is Nested for Loop?

A nested for loop is a for loop placed inside another for loop. The inner loop executes completely for each iteration of the outer loop.

When to Use Nested for Loop?

Nested for loops are used when a task requires repeated processing inside another repetition, such as working with tables, matrices, or pattern generation.

Syntax of Nested for Loop


for outer_variable in sequence:
    for inner_variable in sequence:
        statements

Example: Nested for Loop


for i in range(1,4):
    for j in range(1,3):
        print(i, j)

Output

1 1
1 2
2 1
2 2
3 1
3 2

Important Points

✔ Outer loop controls the main iteration.
✔ Inner loop runs fully for each outer loop cycle.
✔ Proper indentation is required in nested loops.
✔ Too many nested loops reduce readability.
