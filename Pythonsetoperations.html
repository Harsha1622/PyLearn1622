Set Operations in Python

What are Set Operations?

Python sets support mathematical operations such as union, intersection, difference, and symmetric difference. These operations help compare and combine sets.

Union

The union() method returns a new set containing all unique elements from both sets. 


A = {1, 2, 3}
B = {3, 4, 5}

print(A.union(B))
{1, 2, 3, 4, 5}

Intersection

The intersection() method returns the common elements between two sets. 


A = {1, 2, 3}
B = {2, 3, 4}

print(A.intersection(B))
{2, 3}

Difference

The difference() method returns elements present in the first set but not in the second. 


A = {1, 2, 3}
B = {2, 4}

print(A.difference(B))
{1, 3}

Symmetric Difference

The symmetric_difference() method returns elements that are in either set but not in both. 


A = {1, 2, 3}
B = {3, 4, 5}

print(A.symmetric_difference(B))
{1, 2, 4, 5}

Important Points

✔ Union combines elements from both sets.
✔ Intersection returns common elements.
✔ Difference returns unique elements from one set.
✔ Symmetric difference returns non-common elements.
