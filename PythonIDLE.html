Python IDLE & Program Editor

What is IDLE?

IDLE stands for Integrated Development and Learning Environment. It is the default programming environment that comes with Python. IDLE allows users to write, edit, run, and debug Python programs easily.

Python Shell (Interactive Mode)

>> print("Hello")
Hello
The Python shell executes commands immediately and displays the output after each instruction.

Program Editor

The program editor in IDLE is used to write and save Python programs in files. These files can later be executed multiple times without rewriting the code.

Example Program

print("Welcome to Python")
Welcome to Python
