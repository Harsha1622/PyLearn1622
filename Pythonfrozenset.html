Frozen Set in Python

What is a Frozen Set?

A frozenset is an immutable version of a set in Python. Once a frozenset is created, its elements cannot be changed, added, or removed. It behaves like a normal set but does not allow modification.

Creating a Frozen Set

A frozen set is created using the frozenset() function. 


numbers = frozenset([10, 20, 30, 40])

print(numbers)
frozenset({10, 20, 30, 40})

Example: Immutable Nature

Since frozensets are immutable, elements cannot be added or removed. 


numbers = frozenset([1, 2, 3])

numbers.add(4)   # This will raise an error

Frozen Set Operations

Frozen sets support set operations such as union, intersection, difference, and symmetric difference. 


A = frozenset([1, 2, 3])
B = frozenset([3, 4, 5])

print(A.union(B))
frozenset({1, 2, 3, 4, 5})

Important Points

✔ Frozensets are immutable.
✔ They are created using the frozenset() function.
✔ Elements cannot be added or removed.
✔ They support set operations.
