Built-in Functions in Python

What are Built-in Functions?

Built-in functions are predefined functions provided by Python. These functions are always available and can be used directly without importing any module.

Commonly Used Built-in Functions

print(), input(), len(), type(), max(), min(), sum(), abs()

print()


print("Python Programming")

input()


name = input("Enter your name: ")
print(name)

len()


text = "PyLearn"
print(len(text))
7

type()


x = 25
print(type(x))
<class 'int'>

max() and min()


numbers = [10, 45, 5, 80]
print(max(numbers))
print(min(numbers))
80
5

sum()


values = [10, 20, 30]
print(sum(values))
60

abs()


print(abs(-15))
15
