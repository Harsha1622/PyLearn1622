Python Packages

What is a Package?

A package in Python is a collection of modules organized in a directory. Packages help structure large Python projects by grouping related modules together.

Why Use Packages?

✔ Organize large programs into smaller parts.
✔ Avoid naming conflicts between modules.
✔ Improve code readability and maintainability.
✔ Allow reuse of modules across different programs.

Creating a Package

A package is created by making a folder that contains Python modules. The folder usually contains a special file called __init__.py. 


mypackage/
│
├── __init__.py
├── module1.py
└── module2.py

Importing a Module from a Package


from mypackage import module1

module1.function_name()

Example


from math import sqrt

print(sqrt(16))
4.0

Important Points

✔ A package is a collection of modules.
✔ It helps organize large Python programs.
✔ Modules inside packages can be imported using the import statement.
✔ Packages improve code reuse and structure.
