Importing Modules in Python

What is Importing a Module?

Importing a module means using the functions, variables, or classes defined in another Python file. Python provides the import statement to include modules in a program.

1. Import Entire Module


import math

print(math.sqrt(16))
4.0

The entire module is imported and accessed using the module name.

2. Import Specific Function


from math import sqrt

print(sqrt(25))
5.0

Only the required function is imported from the module.

3. Import with Alias


import math as m

print(m.sqrt(36))
6.0

An alias provides a shorter name for the module.

4. Import All Functions


from math import *

print(sqrt(49))
7.0

All functions from the module are imported.

Important Points

✔ Modules allow code reuse.
✔ The import keyword is used to include modules.
✔ Specific functions can be imported using from.
✔ Aliases simplify long module names.
