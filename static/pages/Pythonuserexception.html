User Defined Exceptions in Python

What is a User Defined Exception?

Python allows programmers to create their own exceptions. These are called User Defined Exceptions. They are useful when a program needs to raise specific errors based on custom conditions.

Creating a User Defined Exception

To create a user defined exception, we define a new class that inherits from the built-in Exception class. 


class MyError(Exception):
    pass

Example


class AgeError(Exception):
    pass

age = int(input("Enter age: "))

if age < 18:
    raise AgeError("Age must be 18 or above")
else:
    print("Eligible")
AgeError: Age must be 18 or above

Handling User Defined Exceptions


class AgeError(Exception):
    pass

try:
    age = int(input("Enter age: "))

    if age < 18:
        raise AgeError

    print("Eligible")

except AgeError:
    print("Custom exception occurred")

Important Points

✔ User defined exceptions are created using classes.
✔ They inherit from the built-in Exception class.
✔ The raise statement is used to trigger the exception.
✔ Custom exceptions help control program errors better.
