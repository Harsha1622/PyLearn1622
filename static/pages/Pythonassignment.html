Tuple Packing and Unpacking

Tuple Packing

Tuple packing refers to the process of placing multiple values into a single tuple. When values are separated by commas, Python automatically creates a tuple. 


numbers = (10, 20, 30)

print(numbers)
(10, 20, 30)

Tuple Unpacking

Tuple unpacking is the process of assigning tuple elements to individual variables. Each variable receives one value from the tuple. 


numbers = (10, 20, 30)

a, b, c = numbers

print(a)
print(b)
print(c)
10
20
30

Example: Swapping Variables using Tuple

Python allows swapping two variables easily using tuple unpacking. 


x = 5
y = 10

x, y = y, x

print(x)
print(y)
10
5

Important Points

✔ Tuple packing groups multiple values into one tuple.
✔ Tuple unpacking assigns tuple values to variables.
✔ The number of variables must match the number of tuple elements.
✔ Tuple unpacking is commonly used for swapping values.
