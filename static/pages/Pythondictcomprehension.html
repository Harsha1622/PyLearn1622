Dictionary Comprehension in Python

What is Dictionary Comprehension?

Dictionary comprehension is a short and efficient way to create a dictionary using a single line of code. It is similar to list comprehension but creates key–value pairs.

Syntax


{key : value for item in iterable}

Example: Creating Dictionary


numbers = {x: x*x for x in range(5)}

print(numbers)
{0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

Example with Condition


even_numbers = {x: x*x for x in range(10) if x % 2 == 0}

print(even_numbers)
{0: 0, 2: 4, 4: 16, 6: 36, 8: 64}

Example: Converting List to Dictionary


keys = ["a","b","c"]

dictionary = {key: len(key) for key in keys}

print(dictionary)
{'a': 1, 'b': 1, 'c': 1}

Important Points

✔ Dictionary comprehension creates dictionaries in one line.
✔ Uses curly braces { }.
✔ Can include conditions.
✔ Makes code shorter and cleaner.
