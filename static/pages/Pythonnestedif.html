Nested if Statement in Python

What is Nested if Statement?

A nested if statement is an if statement placed inside another if statement. It is used when one condition depends on another condition.

When to Use Nested if?

Nested if statements are used when multiple conditions must be checked in sequence and the inner condition should execute only when the outer condition is true.

Syntax of Nested if Statement


if condition1:
    if condition2:
        statements

Example: Nested if Statement


age = 25
has_id = True

if age >= 18:
    if has_id:
        print("Entry allowed")

Output

Entry allowed

Important Points

✔ Inner if executes only when the outer condition is true.
✔ Proper indentation is mandatory in Python.
✔ Nested if allows more detailed decision making.
✔ Excessive nesting can reduce code readability.
