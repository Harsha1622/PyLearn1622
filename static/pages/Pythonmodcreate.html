Creation of Modules in Python

What is a Module?

A module is a file that contains Python code such as functions, variables, and classes. Modules help organize programs into separate files and make code reusable.

Why Use Modules?

✔ To organize large programs
✔ To reuse code in multiple programs
✔ To improve readability and maintainability
✔ To divide a program into smaller parts

Creating a Module

A module is simply a Python file with a .py extension. We write functions or variables in the file and save it. 


# file: mymodule.py

def greet(name):
    print("Hello", name)

def add(a, b):
    return a + b

Using the Module

After creating a module, it can be used in another Python program by importing it. 


import mymodule

mymodule.greet("Alex")

result = mymodule.add(10, 20)
print(result)

✔ Easy maintenance
✔ Reduces program complexity
