Dictionary Methods in Python

What are Dictionary Methods?

Dictionary methods are built-in functions used to perform operations such as retrieving keys, values, and key–value pairs from a dictionary. These methods make dictionary handling easier and more efficient.

keys()

The keys() method returns all the keys present in the dictionary. 


student = {
    "name": "Rahul",
    "age": 20
}

print(student.keys())
dict_keys(['name', 'age'])

values()

The values() method returns all the values in the dictionary. 


student = {
    "name": "Rahul",
    "age": 20
}

print(student.values())
dict_values(['Rahul', 20])

items()

The items() method returns all key–value pairs in the dictionary. 


student = {
    "name": "Rahul",
    "age": 20
}

print(student.items())
dict_items([('name', 'Rahul'), ('age', 20)])

copy()

The copy() method creates a copy of the dictionary. 


student = {
    "name": "Rahul",
    "age": 20
}

new_student = student.copy()

print(new_student)

clear()

The clear() method removes all elements from the dictionary. 


student = {
    "name": "Rahul",
    "age": 20
}

student.clear()

print(student)
{}

Important Points

✔ keys() returns all dictionary keys.
✔ values() returns all dictionary values.
✔ items() returns key–value pairs.
✔ copy() creates a duplicate dictionary.
✔ clear() removes all elements.
