Polymorphism in Python

What is Polymorphism?

Polymorphism means "many forms". In Object-Oriented Programming, polymorphism allows the same method name to behave differently depending on the object that calls it.

Why Polymorphism is Used?

✔ Allows one interface to be used for different data types
✔ Improves flexibility in programs
✔ Makes code reusable and scalable

Example: Function Polymorphism


print(len("Python"))
print(len([10,20,30,40]))
6
4

The same function len() works with different data types.

Example: Method Polymorphism


class Dog:
    def sound(self):
        print("Dog barks")

class Cat:
    def sound(self):
        print("Cat meows")

d = Dog()
c = Cat()

d.sound()
c.sound()
Dog barks
Cat meows

Operator Polymorphism


print(5 + 3)
print("Py" + "Learn")
8
PyLearn

The + operator performs addition for numbers and concatenation for strings.

Important Points

✔ Polymorphism means one interface with many behaviors.
✔ Same function can work with different data types.
✔ Python supports polymorphism through functions, methods, and operators.
