Updating Dictionary in Python

Updating Dictionary Values

Dictionary values can be modified by assigning a new value to an existing key. If the key already exists, its value will be updated.

Example: Updating a Value


student = {
    "name": "Rahul",
    "age": 20
}

student["age"] = 21

print(student)
{'name': 'Rahul', 'age': 21}

Updating Multiple Values using update()

The update() method is used to modify one or more key–value pairs at once. 


student = {
    "name": "Rahul",
    "age": 20
}

student.update({"age": 22, "course": "Python"})

print(student)
{'name': 'Rahul', 'age': 22, 'course': 'Python'}

Example: Updating from Another Dictionary


student = {"name":"Rahul","age":20}
new_data = {"age":23,"course":"AI"}

student.update(new_data)

print(student)
{'name': 'Rahul', 'age': 23, 'course': 'AI'}

Important Points

✔ Existing dictionary values can be modified using keys.
✔ Assigning a new value replaces the old value.
✔ update() can modify multiple elements.
✔ update() can also merge another dictionary.
