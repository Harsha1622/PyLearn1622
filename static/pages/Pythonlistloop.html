Looping Through a List in Python

What is Looping Through a List?

Looping through a list means accessing each element of the list one by one. Loops allow us to process or display all elements in a list efficiently.

Using for Loop

The for loop is the most common way to iterate through list elements. 


numbers = [10, 20, 30, 40]

for num in numbers:
    print(num)
10
20
30
40

Using range() with Index

The range() function can be used to access list elements using their index positions. 


numbers = [10, 20, 30, 40]

for i in range(len(numbers)):
    print(numbers[i])
10
20
30
40

Using while Loop

A while loop can also be used to iterate through list elements. 


numbers = [10, 20, 30, 40]

i = 0

while i < len(numbers):
    print(numbers[i])
    i += 1
10
20
30
40

Important Points

✔ Loops help process all list elements easily.
✔ The for loop is commonly used for lists.
✔ The range() function helps access elements using index.
✔ The while loop can also iterate through lists.
