Creation of Sets in Python

Creating a Set

A set can be created by placing elements inside curly braces { } separated by commas. Each element in a set must be unique.

Syntax


set_name = {value1, value2, value3}

Example: Creating a Set


numbers = {10, 20, 30, 40}

print(numbers)
{10, 20, 30, 40}

Creating an Empty Set

An empty set cannot be created using { } because it creates an empty dictionary. To create an empty set, the set() function must be used. 


empty_set = set()

print(empty_set)
set()

Creating a Set from a List


numbers = set([1, 2, 3, 4, 4])

print(numbers)
{1, 2, 3, 4}

Important Points

✔ Sets are created using curly braces { }.
✔ Duplicate elements are automatically removed.
✔ An empty set must be created using set().
✔ Sets can also be created from lists, tuples, or strings.
