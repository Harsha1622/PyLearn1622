List Comprehension in Python

What is List Comprehension?

List comprehension is a short and efficient way to create lists in Python. It allows generating a new list using a single line of code.

Syntax of List Comprehension


new_list = [expression for item in iterable]

✔ expression – operation performed on each element
✔ item – variable representing each element
✔ iterable – list or sequence being processed

Example: Creating a List of Squares


numbers = [1, 2, 3, 4, 5]

squares = [x*x for x in numbers]

print(squares)
[1, 4, 9, 16, 25]

Example: List Comprehension with Condition


numbers = [1, 2, 3, 4, 5, 6]

even_numbers = [x for x in numbers if x % 2 == 0]

print(even_numbers)
[2, 4, 6]

Example: Converting Words to Uppercase


words = ["python", "list", "code"]

upper_words = [word.upper() for word in words]

print(upper_words)
['PYTHON', 'LIST', 'CODE']

Advantages of List Comprehension

✔ Makes code shorter and cleaner.
✔ Improves readability.
✔ Faster than traditional loops in many cases.
✔ Useful for creating lists with conditions.

Important Points

✔ Used to generate lists efficiently.
✔ Can include conditions using if.
✔ Replaces multiple lines of loop code.
✔ Commonly used in Python data processing.
