List as Function Parameter in Python

What is List as Function Parameter?

In Python, lists can be passed as arguments to functions. This allows functions to process multiple values stored inside a list.

Example: Passing List to Function


def show_list(items):
    for item in items:
        print(item)

numbers = [10, 20, 30, 40]

show_list(numbers)
10
20
30
40

Example: Calculating Sum of List Elements


def total(values):
    return sum(values)

numbers = [5, 10, 15]

result = total(numbers)

print(result)
30

Modifying List Inside Function


def add_value(lst):
    lst.append(100)

numbers = [10, 20, 30]

add_value(numbers)

print(numbers)
[10, 20, 30, 100]

Since lists are mutable, changes inside the function affect the original list.

Important Points

✔ Lists can be passed as arguments to functions.
✔ Functions can read or modify list elements.
✔ Lists are mutable, so changes affect the original list.
✔ Functions help process list data efficiently.
