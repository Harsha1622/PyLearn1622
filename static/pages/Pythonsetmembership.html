Membership Testing in Python Sets

What is Membership Testing?

Membership testing is used to check whether a specific element exists in a set. Python provides the in and not in operators for this purpose.

Using the in Operator

The in operator returns True if the element exists in the set. 


numbers = {10, 20, 30, 40}

print(20 in numbers)
True

Using the not in Operator

The not in operator returns True if the element does not exist in the set. 


numbers = {10, 20, 30, 40}

print(50 not in numbers)
True

Important Points

✔ Membership testing checks whether an element exists in a set.
✔ The in operator checks presence.
✔ The not in operator checks absence.
✔ Sets are very efficient for membership testing.
