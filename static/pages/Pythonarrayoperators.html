Array Operators in Python

What are Array Operators?

Array operators are used to perform operations on arrays such as combining arrays, repeating elements, and checking membership.

1. Concatenation Operator (+)

The + operator is used to combine two arrays. 


import array

a = array.array('i', [1, 2, 3])
b = array.array('i', [4, 5, 6])

c = a + b
print(c)
array('i', [1, 2, 3, 4, 5, 6])

2. Repetition Operator (*)

The * operator repeats the array elements. 


import array

a = array.array('i', [2, 4, 6])

print(a * 2)
array('i', [2, 4, 6, 2, 4, 6])

3. Membership Operator (in)

The in operator checks whether an element exists in the array. 


import array

a = array.array('i', [10, 20, 30])

print(20 in a)
True

4. Non-membership Operator (not in)

The not in operator checks if an element is not present in the array. 


import array

a = array.array('i', [10, 20, 30])

print(40 not in a)
True

Important Points

✔ + operator combines arrays.
✔ * operator repeats elements.
✔ in operator checks membership.
✔ not in checks absence of elements.
