Tuple Comprehension in Python

What is Tuple Comprehension?

Tuple comprehension is a concise way of creating tuples using a single line of code. It is similar to list comprehension but uses parentheses instead of square brackets.

Syntax


(expression for item in iterable)

The expression is evaluated for each element in the iterable and the results are stored in a tuple.

Example: Creating a Tuple using Comprehension


numbers = tuple(i for i in range(5))

print(numbers)
(0, 1, 2, 3, 4)

Example with Condition


even_numbers = tuple(i for i in range(10) if i % 2 == 0)

print(even_numbers)
(0, 2, 4, 6, 8)

Important Points

✔ Tuple comprehension uses parentheses.
✔ It creates tuples in a compact and readable way.
✔ Conditions can be applied to filter elements.
✔ Often used with the tuple() function.
