Set Methods in Python

Common Set Methods

Python provides several built-in methods to work with sets. These methods help add, remove, and manipulate elements efficiently.

add()

The add() method inserts a single element into a set. 


numbers = {1, 2, 3}

numbers.add(4)

print(numbers)
{1, 2, 3, 4}

update()

The update() method adds multiple elements to a set. 


numbers = {1, 2}

numbers.update([3, 4, 5])

print(numbers)
{1, 2, 3, 4, 5}

clear()

The clear() method removes all elements from the set. 


numbers = {1, 2, 3}

numbers.clear()

print(numbers)
set()

copy()

The copy() method creates a shallow copy of a set. 


A = {1, 2, 3}

B = A.copy()

print(B)
{1, 2, 3}

Important Points

✔ add() inserts one element.
✔ update() adds multiple elements.
✔ clear() removes all elements from the set.
✔ copy() creates a duplicate set.
