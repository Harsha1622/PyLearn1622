List Slicing in Python

What is List Slicing?

List slicing is used to access a specific range of elements from a list. It allows us to extract a portion of a list without modifying the original list.

Syntax of List Slicing


list[start : stop : step]

✔ start – starting index
✔ stop – ending index (not included)
✔ step – interval between elements

Example: Basic Slicing


numbers = [10, 20, 30, 40, 50]

print(numbers[1:4])
[20, 30, 40]

Example: Slicing from Beginning


numbers = [10, 20, 30, 40]

print(numbers[:3])
[10, 20, 30]

Example: Slicing to End


numbers = [10, 20, 30, 40]

print(numbers[2:])
[30, 40]

Example: Using Step


numbers = [1, 2, 3, 4, 5, 6]

print(numbers[0:6:2])
[1, 3, 5]

Example: Negative Index Slicing


numbers = [10, 20, 30, 40]

print(numbers[-3:-1])
[20, 30]

Important Points

✔ Slicing extracts part of a list.
✔ The stop index is not included in the result.
✔ Step value controls the interval between elements.
✔ Negative indices allow slicing from the end.
