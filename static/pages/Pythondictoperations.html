Dictionary Operations in Python

What are Dictionary Operations?

Dictionary operations are used to perform various tasks such as checking elements, counting items, and iterating through key–value pairs. These operations help in manipulating and managing dictionary data.

Finding Length of Dictionary

The len() function returns the number of key–value pairs in a dictionary. 


student = {
    "name": "Rahul",
    "age": 20,
    "course": "Python"
}

print(len(student))
3

Checking if Key Exists

The in operator checks whether a key exists in the dictionary. 


student = {
    "name": "Rahul",
    "age": 20
}

print("name" in student)
True

Looping Through Dictionary


student = {
    "name": "Rahul",
    "age": 20,
    "course": "Python"
}

for key in student:
    print(key)

Looping Through Key–Value Pairs


student = {
    "name": "Rahul",
    "age": 20
}

for key, value in student.items():
    print(key, value)

Important Points

✔ len() returns number of elements in dictionary.
✔ in operator checks key existence.
✔ Dictionary elements can be accessed using loops.
✔ items() returns key–value pairs for iteration.
