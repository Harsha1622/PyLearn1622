Python Basics

Identifiers

Identifiers are names used to identify variables, functions, classes, modules, or other objects in a Python program. 


total_score = 85
user_name = "admin"

Rules for Identifiers:
✔ Letters, digits, and underscore (_) are allowed
✔ Must start with a letter or underscore
✔ Cannot start with a digit
✔ No special symbols except underscore
✔ Python keywords cannot be used 


2total      # Invalid
total_2     # Valid
total@value # Invalid

Keywords

Keywords are reserved words in Python that have predefined meanings. They cannot be used as identifiers. 


if user_logged_in:
    print("Access granted")

Variables

Variables store data values. Python automatically determines the data type when a value is assigned. 


user_id = 1024
username = "developer"
account_balance = 2500.75


status = "active"

Indentation

Indentation defines blocks of code in Python. Instead of braces, Python uses indentation to organize statements. The standard indentation is four spaces. 


if account_balance > 0:
    print("Transaction allowed")
    print("Processing payment")

Comments

Comments explain the code and improve readability. They are ignored during execution. 


# Display welcome message
print("Welcome to PyLearn")

This program checks user access
and displays a message
'''
