File Operations in Python

Reading from a File

Python provides different methods to read data from a file. The most commonly used methods are read(), readline(), and readlines(). 


file = open("data.txt","r")
content = file.read()
print(content)
file.close()
Displays the content of the file

Writing to a File

The write() method is used to write data into a file. If the file does not exist, Python will create it. 


file = open("data.txt","w")
file.write("Welcome to Python")
file.close()

Appending Data to a File

The append mode (a) allows new data to be added to the end of an existing file without removing the previous content. 


file = open("data.txt","a")
file.write(" Learning File Handling")
file.close()

Closing a File

The close() method is used to close a file after performing operations. Closing the file frees system resources. 


file = open("data.txt","r")
print(file.read())
file.close()

Important Points

✔ read() reads the entire file content.
✔ write() writes data to a file.
✔ append mode adds data without deleting existing content.
✔ close() should always be used after file operations.
