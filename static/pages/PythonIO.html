Input and Output Statements in Python

Output Statement

Output statements display information to the user. Python uses the print() function to display text, numbers, or results of expressions.

Sample Program: Display Output


print("Welcome to Python")
print(10 + 20)
Welcome to Python
30

Input Statement

Input statements allow a program to receive data from the user. Python uses the input() function for this purpose. The value entered is stored as a string by default.

Sample Program: User Input


name = input("Enter your name: ")
print(name)

Formatted Output


name = input("Enter your name: ")
age = int(input("Enter your age: "))

print(f"Name: {name}")
print(f"Age: {age}")
Name: Alex
Age: 21
