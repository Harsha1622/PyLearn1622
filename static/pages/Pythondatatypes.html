Data Types in Python

What is a Data Type?

A data type defines the type of value that a variable can store. Python automatically determines the data type based on the value assigned to a variable, so explicit declaration is not required.

1. Numeric Data Types

Numeric data types store numbers such as integers, floating-point numbers, and complex numbers. 


a = 10      # integer
b = 3.5     # float
c = 2 + 3j  # complex number

2. Sequence Data Types

Sequence data types store ordered collections of values. Examples include strings, lists, and tuples. 


lst = [1, 2, 3]
tup = (10, 20)
name = "Python"

3. Set Data Type

Sets store unordered and unique elements. Duplicate values are automatically removed. 


s = {1, 2, 2, 3}
print(s)

4. Mapping Data Type

Dictionaries store data as key–value pairs. Each key uniquely identifies its associated value. 


student = {"name": "Harsha", "language": "Python"}

5. Boolean Data Type

Boolean values represent logical conditions. They can only be True or False. 


x = True

6. None Data Type

The None type represents the absence of a value. It is commonly used to initialize variables before assigning data. 


value = None
