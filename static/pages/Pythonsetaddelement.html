Adding Elements to a Set in Python

Adding Elements to a Set

Elements can be added to a set using the add() method. This method inserts a single element into the set.

Syntax


set_name.add(element)

Example: Using add()


numbers = {10, 20, 30}

numbers.add(40)

print(numbers)
{10, 20, 30, 40}

Adding Multiple Elements

Multiple elements can be added using the update() method. 


numbers = {10, 20}

numbers.update([30, 40, 50])

print(numbers)
{10, 20, 30, 40, 50}

Important Points

✔ add() inserts a single element into a set.
✔ update() adds multiple elements.
✔ Duplicate elements are ignored.
✔ Sets automatically maintain unique values.
