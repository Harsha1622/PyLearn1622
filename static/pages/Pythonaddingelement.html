Adding Elements to a Dictionary

Adding New Elements

New elements can be added to a dictionary by specifying a new key–value pair. If the key does not exist, Python will create a new entry in the dictionary.

Example: Adding a New Element


student = {
    "name": "Rahul",
    "age": 20
}

student["course"] = "Python"

print(student)
{'name': 'Rahul', 'age': 20, 'course': 'Python'}

Adding Multiple Elements

Multiple elements can be added one by one by assigning new keys. 


student = {}

student["name"] = "Anita"
student["age"] = 22
student["course"] = "Data Science"

print(student)
{'name': 'Anita', 'age': 22, 'course': 'Data Science'}

Using update() Method

The update() method can also add multiple key–value pairs. 


student = {
    "name": "Rahul"
}

student.update({"age": 21, "course": "Python"})

print(student)
{'name': 'Rahul', 'age': 21, 'course': 'Python'}

Important Points

✔ New elements are added using dictionary[key] = value.
✔ If the key does not exist, a new pair is created.
✔ update() can add multiple key–value pairs.
✔ Dictionaries automatically expand when new elements are added.
