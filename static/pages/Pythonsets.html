Sets in Python

What is a Set?

A set is an unordered collection of unique elements in Python. Sets are used to store multiple values in a single variable without duplicates. They are useful for mathematical operations such as union, intersection, and difference.

Characteristics of Sets

✔ Sets are unordered collections.
✔ Sets do not allow duplicate values.
✔ Sets are mutable (elements can be added or removed).
✔ Sets are defined using curly braces { }.

Syntax of Set


set_name = {value1, value2, value3}

Example: Creating a Set


numbers = {10, 20, 30, 40}

print(numbers)
{10, 20, 30, 40}

Example: Duplicate Values


numbers = {1, 2, 2, 3, 4}

print(numbers)
{1, 2, 3, 4}

Important Points

✔ Duplicate elements are automatically removed.
✔ Sets are unordered, so indexing is not supported.
✔ Sets are useful for membership testing and mathematical operations.
✔ Sets can store different data types.
