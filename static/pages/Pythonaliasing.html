Aliasing in Python Lists

What is Aliasing?

Aliasing occurs when two variables refer to the same list object in memory. Instead of creating a new list, Python creates another reference to the same list.

Example of Aliasing


list1 = [10, 20, 30]

list2 = list1

print(list1)
print(list2)
[10, 20, 30]
[10, 20, 30]

Effect of Aliasing

If one variable modifies the list, the change will also appear in the other variable, because both variables refer to the same object. 


list1 = [10, 20, 30]

list2 = list1

list2[1] = 50

print(list1)
print(list2)
[10, 50, 30]
[10, 50, 30]

Why Aliasing Happens?

✔ Python variables store references to objects.
✔ Assigning one list variable to another creates a reference.
✔ No new list is created during simple assignment.

Important Points

✔ Aliasing means two variables refer to the same list.
✔ Changes through one variable affect the other.
✔ Aliasing does not create a new list.
✔ To create a separate copy, list cloning is required.
