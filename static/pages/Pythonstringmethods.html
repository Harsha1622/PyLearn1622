String Methods in Python

What are String Methods?

String methods are built-in functions used to perform operations on strings such as changing case, replacing text, removing spaces, and splitting strings.

1. upper()

The upper() method converts all characters in a string to uppercase letters. 


text = "python"

print(text.upper())
PYTHON

2. lower()

The lower() method converts all characters in a string to lowercase letters. 


text = "PYTHON"

print(text.lower())
python

3. replace()

The replace() method replaces a word or character with another value. 


text = "Hello Python"

print(text.replace("Python", "World"))
Hello World

4. split()

The split() method divides a string into a list of words. 


text = "Python is easy"

print(text.split())
['Python', 'is', 'easy']

5. strip()

The strip() method removes extra spaces from the beginning and end of a string. 


text = "   Python   "

print(text.strip())
Python

6. find()

The find() method returns the position of a substring in a string. 


text = "Hello Python"

print(text.find("Python"))
6

Important Points

✔ upper() converts text to uppercase.
✔ lower() converts text to lowercase.
✔ replace() replaces words or characters.
✔ split() divides a string into a list.
✔ strip() removes extra spaces.
✔ find() returns the position of a word.
