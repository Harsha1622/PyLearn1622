Lambda with map(), filter(), and reduce()

Using Lambda with Built-in Functions

Lambda functions are commonly used with functions like map(), filter(), and reduce() to perform operations on sequences such as lists.

1. map() Function

The map() function applies a given function to each item in a sequence and returns the result. 


numbers = [1, 2, 3, 4]

result = list(map(lambda x: x * 2, numbers))

print(result)
[2, 4, 6, 8]

2. filter() Function

The filter() function selects elements from a sequence based on a condition. 


numbers = [1, 2, 3, 4, 5, 6]

result = list(filter(lambda x: x % 2 == 0, numbers))

print(result)
[2, 4, 6]

3. reduce() Function

The reduce() function performs a cumulative operation on elements of a sequence and returns a single result. It requires the functools module. 


from functools import reduce

numbers = [1, 2, 3, 4]

result = reduce(lambda a, b: a + b, numbers)

print(result)
10

Important Points

✔ map() applies a function to each element.
✔ filter() selects elements based on a condition.
✔ reduce() combines elements into a single value.
✔ Lambda functions make these operations shorter and simpler.
