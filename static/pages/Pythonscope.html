Scope of Variables in Python

What is Variable Scope?

Variable scope refers to the region of a program where a variable can be accessed. In Python, variables can have either local scope or global scope.

1. Local Variables

A local variable is defined inside a function and can be accessed only within that function. 


def show():
    x = 10
    print(x)

show()
10

The variable x exists only inside the function.

2. Global Variables

A global variable is defined outside all functions and can be accessed anywhere in the program. 


x = 20

def show():
    print(x)

show()
20

Using the global Keyword

The global keyword is used to modify a global variable inside a function. 


x = 5

def change():
    global x
    x = 10

change()
print(x)
10

Important Points

✔ Local variables exist only inside functions.
✔ Global variables can be accessed throughout the program.
✔ The global keyword allows modification of global variables inside functions.
