Tuple Methods in Python

Tuple Methods

Unlike lists, tuples are immutable. Because of this, Python provides only a few built-in methods for tuples. The two commonly used tuple methods are count() and index().

count() Method

The count() method returns the number of times a specified value appears in a tuple. 


numbers = (1, 2, 3, 2, 4, 2)

print(numbers.count(2))
3

index() Method

The index() method returns the index of the first occurrence of a specified element in a tuple. 


numbers = (10, 20, 30, 40)

print(numbers.index(30))
2

Important Points

✔ Tuples are immutable, so methods that modify data are not available.
✔ count() returns how many times an element appears.
✔ index() returns the position of the first matching element.
✔ If the element is not found, index() raises a ValueError.
