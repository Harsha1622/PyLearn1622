Removing Elements from Dictionary

Removing Dictionary Elements

Elements in a dictionary can be removed using methods like pop(), popitem(), del, and clear().

Using pop()

The pop() method removes the element with the specified key. 


student = {
    "name": "Rahul",
    "age": 20,
    "course": "Python"
}

student.pop("age")

print(student)
{'name': 'Rahul', 'course': 'Python'}

Using popitem()

The popitem() method removes the last inserted key–value pair. 


student = {
    "name": "Rahul",
    "age": 20
}

student.popitem()

print(student)

Using del Keyword


student = {
    "name": "Rahul",
    "age": 20
}

del student["age"]

print(student)
{'name': 'Rahul'}

Using clear()

The clear() method removes all elements from the dictionary. 


student = {
    "name": "Rahul",
    "age": 20
}

student.clear()

print(student)
{}

Important Points

✔ pop() removes a specific key.
✔ popitem() removes the last inserted pair.
✔ del removes a specific key.
✔ clear() removes all elements.
