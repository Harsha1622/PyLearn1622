Opening Files in Python

Opening a File

In Python, a file must be opened before performing any operation such as reading or writing. The open() function is used to open a file.

Syntax


file_object = open("filename", "mode")

Here:
filename → Name of the file
mode → Specifies how the file will be used

File Modes

r → Read mode (default)
w → Write mode (creates new file if not exists)
a → Append mode (adds data at the end)
r+ → Read and write mode

Example


file = open("data.txt", "r")

print(file)
<_io.TextIOWrapper name='data.txt' mode='r' encoding='utf-8'>

Opening File Using with Statement

Python provides the with statement to open files safely. It automatically closes the file after use. 


with open("data.txt","r") as file:
    print(file.read())

Important Points

✔ open() is used to open files.
✔ Different modes control how the file is used.
✔ Always close files after using them.
✔ The with statement automatically closes files.
