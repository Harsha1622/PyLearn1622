Accessing Tuple Elements in Python

Accessing Tuple Elements

Tuple elements can be accessed using their index position. The index starts from 0, meaning the first element has index 0, the second element has index 1, and so on.

Example: Accessing Elements


numbers = (10, 20, 30, 40)

print(numbers[0])
print(numbers[2])
10
30

Accessing Elements Using Negative Index

Python also supports negative indexing. Negative indexes start from the end of the tuple. 


numbers = (10, 20, 30, 40)

print(numbers[-1])
print(numbers[-2])
40
30

Important Points

✔ Indexing starts from 0.
✔ Negative indexing starts from the end.
✔ Tuples allow element access but not modification.
✔ Accessing an invalid index causes an IndexError.
