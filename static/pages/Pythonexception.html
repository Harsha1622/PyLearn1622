Exception Handling in Python

What is an Exception?

An exception is an error that occurs during the execution of a program. When an exception occurs, the program may stop running unless the error is handled properly.

Common Types of Exceptions

Python provides many built-in exceptions. Some common examples include:

✔ ZeroDivisionError – occurs when dividing by zero.
✔ ValueError – occurs when an invalid value is used.
✔ TypeError – occurs when incorrect data types are used.
✔ FileNotFoundError – occurs when a file does not exist.

Example of an Exception


num1 = 10
num2 = 0

result = num1 / num2

print(result)
ZeroDivisionError: division by zero

Why Exception Handling is Important

✔ Prevents program crashes.
✔ Allows the program to continue running.
✔ Helps handle unexpected situations gracefully.
✔ Improves program reliability.
