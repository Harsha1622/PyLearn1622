Inheritance in Python

What is Inheritance?

Inheritance is a feature of Object-Oriented Programming that allows one class to acquire the properties and methods of another class. The class that is inherited from is called the parent class or base class, and the class that inherits is called the child class or derived class.

Why Inheritance is Used?

✔ Promotes code reusability
✔ Reduces code duplication
✔ Makes programs easier to maintain
✔ Represents real-world relationships

Syntax of Inheritance


class Parent:
    # parent class code

class Child(Parent):
    # child class code

Example: Single Inheritance


class Animal:

    def speak(self):
        print("Animal makes sound")

class Dog(Animal):

    def bark(self):
        print("Dog barks")

d = Dog()

d.speak()
d.bark()
Animal makes sound
Dog barks

Types of Inheritance in Python

✔ Single Inheritance
✔ Multiple Inheritance
✔ Multilevel Inheritance
✔ Hierarchical Inheritance
✔ Hybrid Inheritance

Example: Multilevel Inheritance


class A:
    def showA(self):
        print("Class A")

class B(A):
    def showB(self):
        print("Class B")

class C(B):
    def showC(self):
        print("Class C")

obj = C()

obj.showA()
obj.showB()
obj.showC()
Class A
Class B
Class C

Important Points

✔ Inheritance allows code reuse.
✔ Child classes inherit attributes and methods of parent classes.
✔ Python supports multiple types of inheritance.
✔ It helps build hierarchical class relationships.
