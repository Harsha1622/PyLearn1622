while Loop in Python

What is while Loop?

The while loop is a looping statement used to execute a block of code repeatedly as long as a given condition is true. The condition is checked before each iteration.

When to Use while Loop?

The while loop is used when the number of iterations is not known in advance and depends on a condition.

Syntax of while Loop


while condition:
    statements

Example: while Loop


count = 1

while count <= 5:
    print(count)
    count += 1

Output

1
2
3
4
5

Important Points

✔ Condition is checked before each iteration.
✔ If the condition is false initially, the loop does not execute.
✔ Loop variable must be updated to avoid infinite loops.
✔ Proper indentation is required in Python loops.
