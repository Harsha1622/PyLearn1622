Function Arguments in Python

What are Function Arguments?

Function arguments are values passed to a function when it is called. They allow the function to receive data and perform operations on it. Arguments make functions flexible and reusable.

Types of Function Arguments

Python supports the following types of arguments:

✔ Positional Arguments
✔ Keyword Arguments
✔ Default Arguments
✔ Variable-Length Arguments
✔ Keyword Variable-Length Arguments

1. Positional Arguments


def add(a, b):
    print(a + b)

add(10, 20)
30

Values are passed in the same order as parameters.

2. Keyword Arguments


def student(name, age):
    print(name, age)

student(name="Alex", age=21)
Alex 21

Arguments are passed using parameter names.

3. Default Arguments


def greet(name="Student"):
    print("Hello", name)

greet()
Hello Student

A default value is used if no argument is provided.

4. Variable-Length Arguments (*args)


def total(*numbers):
    print(sum(numbers))

total(10, 20, 30)
60

The * symbol allows passing multiple positional values.

5. Keyword Variable-Length Arguments (**kwargs)


def student_info(**data):
    print(data)

student_info(name="Alex", age=21)
{'name': 'Alex', 'age': 21}

The ** symbol allows passing multiple keyword arguments.

Important Points

✔ Arguments pass data to functions.
✔ Positional arguments depend on order.
✔ Keyword arguments use parameter names.
✔ Default arguments provide fallback values.
✔ *args and **kwargs allow flexible input.
