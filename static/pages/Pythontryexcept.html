try and except in Python

What is try-except?

The try and except blocks are used to handle exceptions in Python. The code that may cause an error is placed inside the try block, and the error is handled inside the except block.

Syntax


try:
    statements
except:
    statements

If an exception occurs inside the try block, the control immediately moves to the except block.

Example


try:
    num1 = 10
    num2 = 0
    result = num1 / num2
    print(result)

except:
    print("Error occurred")
Error occurred

Handling Specific Exceptions

Python allows handling specific types of exceptions. 


try:
    x = int("abc")

except ValueError:
    print("Invalid value")
Invalid value

Important Points

✔ try block contains code that may cause an error.
✔ except block handles the error.
✔ Specific exceptions can also be handled.
✔ Programs continue running even after errors.
