if–elif–else Ladder in Python

What is if–elif–else Ladder?

The if–elif–else ladder is used to test multiple conditions. When a condition becomes true, its corresponding block is executed and the remaining conditions are skipped.

When to Use?

This structure is used when multiple conditions exist and only one block should execute.

Syntax


if condition1:
    statements
elif condition2:
    statements
elif condition3:
    statements
else:
    statements

Example


marks = 78

if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
elif marks >= 60:
    print("Grade C")
else:
    print("Fail")

Output

Grade B

Important Points

✔ Conditions are checked from top to bottom
✔ The first true condition executes
✔ The else block is optional
✔ Proper indentation is required
