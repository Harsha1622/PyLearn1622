Accessing Dictionary Elements

Accessing Values in a Dictionary

Dictionary values are accessed using their keys. The key is placed inside square brackets [ ].

Example: Accessing Value using Key


student = {
    "name": "Rahul",
    "age": 20,
    "course": "Python"
}

print(student["name"])
Rahul

Using get() Method

The get() method is another way to access dictionary values. It avoids errors if the key does not exist. 


student = {
    "name": "Rahul",
    "age": 20
}

print(student.get("age"))
20

Accessing All Keys


student = {
    "name": "Rahul",
    "age": 20
}

print(student.keys())
dict_keys(['name', 'age'])

Accessing All Values


student = {
    "name": "Rahul",
    "age": 20
}

print(student.values())
dict_values(['Rahul', 20])

Accessing Key-Value Pairs


student = {
    "name": "Rahul",
    "age": 20
}

print(student.items())
dict_items([('name', 'Rahul'), ('age', 20)])

Important Points

✔ Dictionary elements are accessed using keys.
✔ Square brackets [ ] retrieve the value of a key.
✔ get() method avoids key errors.
✔ keys() returns all keys.
✔ values() returns all values.
✔ items() returns key-value pairs.
