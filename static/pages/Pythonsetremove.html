Removing Elements from a Set in Python

Removing Elements

Elements can be removed from a set using methods such as remove(), discard(), and pop(). These methods help delete elements from a set.

Using remove()

The remove() method deletes a specified element. If the element is not found, it raises an error. 


numbers = {10, 20, 30, 40}

numbers.remove(20)

print(numbers)
{10, 30, 40}

Using discard()

The discard() method removes an element without raising an error if the element does not exist. 


numbers = {10, 20, 30}

numbers.discard(20)

print(numbers)
{10, 30}

Using pop()

The pop() method removes and returns a random element from the set because sets are unordered. 


numbers = {10, 20, 30}

numbers.pop()

print(numbers)

Important Points

✔ remove() deletes a specific element.
✔ discard() removes an element without error if it is absent.
✔ pop() removes a random element.
✔ clear() removes all elements from the set.
