Jumping Statements in Python

What are Jumping Statements?

Jumping statements alter the normal flow of execution in a program. Python provides break, continue, pass, and assert.

break Statement

The break statement terminates the loop immediately. 


for i in range(1,6):
    if i == 4:
        break
    print(i)
1
2
3

continue Statement

The continue statement skips the current iteration. 


for i in range(1,6):
    if i == 3:
        continue
    print(i)
1
2
4
5

pass Statement

The pass statement does nothing and acts as a placeholder. 


for i in range(3):
    pass

print("Loop completed")
Loop completed

assert Statement

The assert statement is used for debugging and testing conditions. 


marks = 35

assert marks >= 40, "Marks must be 40 or above"

print("Pass")
AssertionError: Marks must be 40 or above

Important Points

✔ break exits the loop immediately.
✔ continue skips the current iteration.
✔ pass acts as an empty placeholder.
✔ assert helps detect logical errors during testing.
