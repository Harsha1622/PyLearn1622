Classes in Python

What is a Class?

A class is a blueprint used to create objects. It defines the properties (variables) and behaviors (methods) that objects created from the class will have.

Why Use Classes?

✔ Helps organize large programs
✔ Groups related data and functions together
✔ Allows creation of multiple objects with similar properties
✔ Supports Object-Oriented Programming concepts

Syntax of a Class


class ClassName:
    # attributes
    # methods

Classes are defined using the class keyword.

Example: Simple Class


class Student:
    name = "Alex"

    def display(self):
        print("Student Name:", self.name)

s = Student()
s.display()
Student Name: Alex

Important Points

✔ A class is a blueprint for objects.
✔ Classes are defined using the class keyword.
✔ Methods are functions defined inside a class.
✔ The self parameter refers to the current object.
