File Handling in Python

What is File Handling?

File handling in Python is used to store and retrieve data from files. It allows programs to read data from files and write data into files. This helps in saving information permanently instead of storing it only in memory.

Why File Handling is Important

✔ Store data permanently.
✔ Read large amounts of data from files.
✔ Write program results to files.
✔ Manage data outside the program.

Types of Files in Python

There are mainly two types of files:

Text Files – Store data as plain text.
Binary Files – Store data in binary format.

Basic Steps in File Handling

File handling usually involves four steps:

✔ Opening a file
✔ Reading or writing data
✔ Processing the data
✔ Closing the file

Example


file = open("data.txt","r")

print(file.read())

file.close()
Contents of data.txt will be displayed

Important Points

✔ Files must be opened before reading or writing.
✔ Always close files after using them.
✔ Python provides different modes for file operations.
✔ File handling improves data storage and management.
