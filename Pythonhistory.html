Introduction to Python

About Python Language

Python is a high-level, object-oriented, interpreted scripting language. It executes instructions line by line using an interpreter without the need for a compiler.

Programming Style

Python supports both Procedural Oriented Programming (POP) and Object Oriented Programming (OOP).

History of Python

Python was developed by Guido van Rossum. The name was inspired by Monty Python's Flying Circus.

Sample Program

print("Hello, World!")

Output

Hello, World!
