String Modules in Python

What are String Modules?

Python provides a built-in string module that contains useful constants and functions for working with text data. It helps perform common string operations easily.

Importing the String Module


import string

The string module must be imported before using its features.

1. string.ascii_letters

Returns all lowercase and uppercase letters. 


import string

print(string.ascii_letters)
abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ

2. string.digits

Returns all numeric characters from 0 to 9. 


import string

print(string.digits)
0123456789

3. string.punctuation

Returns all punctuation characters. 


import string

print(string.punctuation)
!"#$%&'()*+,-./:;<=>?@[\]^_`{|}~

4. string.whitespace

Returns whitespace characters such as spaces, tabs, and new lines. 


import string

print(string.whitespace)
(space, tab, newline etc.)

Important Points

✔ The string module provides useful constants.
✔ It helps in handling text data efficiently.
✔ Common constants include letters, digits, punctuation, and whitespace.
