List Methods in Python

What are List Methods?

List methods are built-in functions that allow us to modify and manage list elements easily. These methods help add, remove, sort, and manipulate data stored inside a list.

1. append()

The append() method adds an element to the end of a list. 


numbers = [1, 2, 3]

numbers.append(4)
print(numbers)
[1, 2, 3, 4]

2. insert()

The insert() method adds an element at a specified position. 


numbers = [10, 20, 30]

numbers.insert(1, 15)
print(numbers)
[10, 15, 20, 30]

3. remove()

The remove() method removes a specific element from a list. 


numbers = [10, 20, 30]

numbers.remove(20)
print(numbers)
[10, 30]

4. pop()

The pop() method removes an element using its index. 


numbers = [10, 20, 30]

numbers.pop(1)
print(numbers)
[10, 30]

5. sort()

The sort() method arranges elements in ascending order. 


numbers = [5, 2, 8, 1]

numbers.sort()
print(numbers)
[1, 2, 5, 8]

6. reverse()

The reverse() method reverses the order of elements. 


numbers = [1, 2, 3]

numbers.reverse()
print(numbers)
[3, 2, 1]

7. count()

The count() method returns the number of occurrences of an element in a list. 


numbers = [1, 2, 2, 3]

print(numbers.count(2))
2

8. index()

The index() method returns the position of an element. 


numbers = [10, 20, 30]

print(numbers.index(20))
1

Important Points

✔ List methods modify list elements.
✔ append() adds element at end.
✔ insert() adds element at specific index.
✔ remove() deletes a specific value.
✔ pop() removes element using index.
✔ sort() and reverse() change order of elements.
