Lambda (Anonymous) Functions in Python

What is a Lambda Function?

A lambda function is a small anonymous function defined using the lambda keyword. It does not have a name and is usually used for short operations that are needed for a short period of time.

Syntax of Lambda Function


lambda arguments : expression

The lambda function can take any number of arguments but can contain only one expression.

Example: Lambda Function


add = lambda a, b: a + b

print(add(10, 20))
30

Example: Square of a Number


square = lambda x: x * x

print(square(5))
25

When to Use Lambda Functions?

✔ When a small function is needed temporarily.
✔ When writing short one-line functions.
✔ Often used with functions like map(), filter(), and reduce().

Important Points

✔ Lambda functions are anonymous functions.
✔ Defined using the lambda keyword.
✔ Can have multiple arguments but only one expression.
✔ Mostly used for short operations.
