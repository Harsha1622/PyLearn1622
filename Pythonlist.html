Introduction to Lists in Python

What is a List?

A list is an ordered collection of elements used to store multiple values in a single variable. Lists are one of the most commonly used data structures in Python.

Characteristics of Lists

✔ Ordered collection
✔ Mutable (can be modified)
✔ Allows duplicate values
✔ Can store different data types

Creating a List


numbers = [10, 20, 30, 40]
print(numbers)
[10, 20, 30, 40]

List with Different Data Types


data = [10, "Python", 3.5, True]
print(data)
[10, 'Python', 3.5, True]

Accessing List Elements


numbers = [10, 20, 30, 40]

print(numbers[0])
print(numbers[2])
10
30

Important Points

✔ Lists are created using square brackets [ ].
✔ Elements are separated by commas.
✔ Index starts from 0.
✔ Lists can store multiple data types.
