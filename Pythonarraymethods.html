Array Methods in Python

What are Array Methods?

Array methods are built-in functions used to perform operations on arrays such as adding elements, removing elements, or finding information about elements.

1. append()

The append() method adds a new element at the end of the array. 


import array

arr = array.array('i', [10, 20, 30])

arr.append(40)

print(arr)
array('i', [10, 20, 30, 40])

2. insert()

The insert() method inserts an element at a specific position. 


import array

arr = array.array('i', [10, 20, 30])

arr.insert(1, 15)

print(arr)
array('i', [10, 15, 20, 30])

3. remove()

The remove() method removes the specified element from the array. 


import array

arr = array.array('i', [10, 20, 30])

arr.remove(20)

print(arr)
array('i', [10, 30])

4. pop()

The pop() method removes and returns the last element of the array. 


import array

arr = array.array('i', [10, 20, 30])

arr.pop()

print(arr)
array('i', [10, 20])

5. index()

The index() method returns the position of an element. 


import array

arr = array.array('i', [10, 20, 30])

print(arr.index(20))
1

6. count()

The count() method returns the number of times an element appears in the array. 


import array

arr = array.array('i', [10, 20, 20, 30])

print(arr.count(20))
2

Important Points

✔ append() adds an element at the end.
✔ insert() adds an element at a specific position.
✔ remove() deletes a specific element.
✔ pop() removes the last element.
✔ index() finds the position of an element.
✔ count() counts the occurrences of an element.
