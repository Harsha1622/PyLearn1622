Functions in Python

Definition of Function

A function is a block of reusable code that performs a specific task. Functions help reduce code repetition, improve readability, and make programs easier to maintain.

Why Functions are Used?

✔ To reuse code multiple times.
✔ To reduce program size.
✔ To make programs modular and structured.
✔ To simplify debugging and testing.

Syntax of a Function


def function_name():
    statements

Sample Program: Simple Function


def greet():
    print("Welcome to Python")

greet()

Output

Welcome to Python
