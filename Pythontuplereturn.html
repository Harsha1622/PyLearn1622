Tuple as Return Value in Python

Returning Multiple Values

A function in Python can return multiple values at once. When multiple values are returned, Python automatically packs them into a tuple.

Example: Returning a Tuple


def get_numbers():
    return 10, 20, 30

result = get_numbers()

print(result)
(10, 20, 30)

Accessing Returned Tuple

The returned tuple can be accessed using indexing or by unpacking it into multiple variables. 


def get_data():
    return "Python", 3.11

name, version = get_data()

print(name)
print(version)
Python
3.11

Important Points

✔ Python functions can return multiple values.
✔ Returned values are automatically packed into a tuple.
✔ Tuple unpacking can be used to store returned values in variables.
✔ This feature improves code readability and efficiency.
