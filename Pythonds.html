Data Structures in Python

What are Data Structures?

Data structures are ways of organizing and storing data so that it can be accessed and modified efficiently. They help manage large amounts of data in programs.

Common Data Structures in Python

Python provides several built-in data structures:

✔ List
✔ Tuple
✔ Set
✔ Dictionary

List

A list is an ordered and mutable collection of elements. Lists allow duplicate values and support indexing and slicing. 


numbers = [10, 20, 30, 40]
print(numbers)
[10, 20, 30, 40]

Tuple

A tuple is an ordered but immutable collection. Once created, tuple elements cannot be modified. 


data = (1, 2, 3)
print(data)
(1, 2, 3)

Set

A set is an unordered collection of unique elements. Duplicate values are automatically removed. 


values = {1, 2, 2, 3}
print(values)
{1, 2, 3}

Dictionary

A dictionary stores data in key–value pairs. Each key is unique and used to access its value. 


student = {"name":"Alex", "age":21}
print(student["name"])
Alex

Why Data Structures are Important

✔ Efficient data storage
✔ Faster data access
✔ Better program organization
✔ Useful for solving complex problems
