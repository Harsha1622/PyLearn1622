Objects in Python

What is an Object?

An object is an instance of a class. When a class is defined, no memory is allocated. Memory is allocated only when an object of that class is created.

Creating an Object


class Student:
    name = "Alex"

s = Student()

Here, s is an object of the class Student.

Accessing Object Attributes


class Student:
    name = "Alex"

s = Student()

print(s.name)
Alex

Example: Object with Method


class Student:

    def greet(self):
        print("Hello Student")

s = Student()
s.greet()
Hello Student

Important Points

✔ An object is created from a class.
✔ Objects represent real-world entities.
✔ Objects can access class attributes and methods.
✔ Multiple objects can be created from a single class.
